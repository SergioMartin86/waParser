# waParser

A Worms Armageddon replay input parser -- it outputs inputs in Bizhawk format

It is based on the replay format logic, as explaned in https://worms2d.info/Replay_file

Most, if not all of the tests included in the tests/ folder are taken from https://worms2d.info/Mission_records. All credits to their authors.
